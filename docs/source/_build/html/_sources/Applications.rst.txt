Applications
============

What are Applications?
~~~~~~~~~~~~~~~~~~~~~~
An Application is an instance of a software application that is made available to citizens to interact with government services. An Application belongs to an :ref:`Organization <organizations>` and may have it's own :ref:`Policy <policies>`. An Application may have more than one :ref:`Service Card <service_cards>` that is presented to the citizen to access the application. An Application may also have a set of :ref:`Email Templates <email_templates>` available that can be managed and consumed by the Application.
